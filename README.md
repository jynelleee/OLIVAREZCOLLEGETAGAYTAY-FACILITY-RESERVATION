TITLE: Olivarez College Tagaytay Facility Reservation System

PROJECT DESCRIPTION: 
This web‑based application allows students and faculty of Olivarez College Tagaytay to easily reserve school facilities — such as open court, classrooms, gyms, etc. — without needing paper forms or manual processes.  
The system aims to replace traditional manual reservation methods with a fast, organized, and user‑friendly online platform, while giving administrators an easy way to monitor and manage reservations.

KEY FEATURES:
- Account creation / Login system — secure user authentication.  
- Facility reservation form — users (students/faculty) can request facility booking.  
- Facility image preview — users can see a preview/image of the facility before reserving.  
- Prevention of duplicate bookings — the system checks to avoid overlapping date & time reservations.  
- Admin dashboard — for administrators to view and manage all reservations.  
- LocalStorage-based database — stores user data and reservations directly in the browser (for now).  
- Clean, modern UI with background images — provides a user-friendly and visually appealing interface.
